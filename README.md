## ✍️ 如何写一篇新笔记

1. 在 `notes/` 中新建一个 `YYYY-MM-DD-标题.md` 文件，首行写标题（建议使用 `# 标题`）。
2. 提交更改（本地执行 `git commit && git push`，或在 GitHub 网页直接 Commit）。
3. GitHub Actions 会自动更新 README 中的文章目录，无需手动运行脚本。
